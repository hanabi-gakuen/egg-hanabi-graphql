# egg-hanabi-graphql

graphql plugin for hanabi

## Install

## Usage

```js
// {app_root}/config/plugin.js
exports.hanabiGraphql = {
  enable: true,
  package: 'egg-hanabi-graphql',
};
```

## Configuration

```js
// {app_root}/config/config.default.js
exports.hanabiGraphql = {
  router: '/graphql',
  graphiql: true
};
```
